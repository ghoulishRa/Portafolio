School Projects

Advanced embedded vehicle steering control design

Designed a vehicle steering control system for a toy car, simulating a steer by wire system. For the project I used a 32-bit microcontroller based on an ARM architecture.

It included several communication protocols to connect different sensors such as Serial, CAN, IIC and SPI. The motor had an absolute encoder sensor to check the wheels position and an IMU to receive and analyze the frequency vibrations in order to identify potential disturbances. An OLED display was used as user interface. Also it had an ESP32 to communicate the system to a mobile dashboard.

FPGA embedded system

Designed an embedded system based on an FPGA to recreate an analog controller for a videogame. The project implemented an 8-bit analog sensor controller to play the classic Atari breakout game on a PC. It has a serial communication protocol to connect the FPGA to an external computer. The lives were shown on a VGA monitor. This project was presented to INTEL.

Electric Current Monitoring System

Designed a system capable of monitoring the current consumption of any appliance to generate an approximate energy expenditure report. The project implemented an 8-bit Atmel microcontroller to design the system and several analog and digital sensors, such as current sensors, a Bluetooth module, a LCD display and a computer to show the report receipt.
